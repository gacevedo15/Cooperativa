# Cooperativa Empresarial

## Descripción del proyecto

El objetivo del proyecto es desarrollar un sistema de gestión para una cooperativa agrícola que permita gestionar de forma eficiente y automatizada las tareas diarias, incluyendo la gestión de los productos, de los productores y clientes, así como el control de inventario, la gestión de pedidos y las estadísticas anuales.

El sistema estará diseñado utilizando los principios de la programación orientada a objetos (POO), para asegurar su escalabilidad, modularidad y facilidad de mantenimiento. Se utilizará el lenguaje de programación Java y se hará uso de la clase Scanner para interactuar con el usuario mediante un menú mostrado por la consola.

El sistema permitirá a los usuarios realizar diversas acciones, entre ellas destacan:

- Gestionar los productos.
- Actualizar la información de los clientes.
- Gestionar los productores.
- Simular pedidos como clientes.
- Gestionar diferentes ofertas logísticas.

En definitiva, el objetivo es crear una plataforma que permita una gestión eficiente y centralizada de las tareas diarias de la cooperativa agrícola, mejorando así la calidad de su servicio y su eficiencia en la gestión de sus recursos.

## Cómo iniciar el proyecto

Para iniciar el proyecto, sigue estos pasos:

1. Compila el código utilizando algún IDE compatible con Java o genera un archivo .jar.
2. Ejecuta el archivo compilado o el .jar en una máquina con Java instalado.
3. Se mostrará un menú en la consola donde podrás seleccionar las diferentes acciones disponibles.

## Instrucciones para el usuario

Una vez que el proyecto esté en funcionamiento, podrás realizar las siguientes acciones:

- Gestionar los productos: podrás agregar, actualizar y eliminar productos del inventario.
- Actualizar información de los clientes: podrás modificar los datos de los clientes, como su nombre, dirección, etc.
- Gestionar los productores: podrás agregar, modificar y eliminar información de los productores asociados a la cooperativa.
- Simular pedidos como clientes: podrás realizar pedidos simulados para probar el funcionamiento del sistema.
- Gestionar ofertas logísticas: podrás administrar las diferentes ofertas logísticas disponibles para los clientes.

Sigue las instrucciones que se mostrarán en la consola para interactuar con el sistema y realizar las diferentes acciones.
